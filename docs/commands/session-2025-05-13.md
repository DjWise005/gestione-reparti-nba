**Causa:** Progetto aveva Prisma 7.8.0 con breaking changes  
**Fix:** Downgrade a Prisma 5.22.0

**Errore:** `Could not find Prisma Schema`  
**Causa:** Comando `cat` malformato cancellò il file  
**Fix:** Ricreato schema.prisma via VS Code
